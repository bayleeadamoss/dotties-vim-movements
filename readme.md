## Vim Movements

Sets the leader key as space ` `.

Move between splits:
* `ctrl+h` - left split
* `ctrl+j` - bottom split
* `ctrl+k` - upper buffer
* `ctrl+l` - right split

Move between buffers:
* `<leader>n` - next buffer in list
* `<leader>n` - previous buffer in list
* `<leader>#` - toggle to last used buffer
